Prework Study Guide

HTML

  • HTML contains the structure of the webpage, can be compared to a frame of a house
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The meta tag defines metadata about an HTML document
  • UTF-8 is a variable-length character encoding used for electronic communication
  • Not all information inside of the HTML file is visible to the webpage user

CSS

  • CSS is a programming language used by developers to define how webpage content is presented to users
  • There are three styles of CSS: Inline, Internal, and External
  • The most common way to style a webpage is by using an Externel CSS
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The * selector applies a rule to all visible elements on the page
  • Selectors can be combined to allow declarations to affect both selectors - DRY: Do Not Repeat Yourself

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file]: adds a file as it looks now to your next commit
  • git commit -m "[descriptive message]": commits your staged content as a new commit snapshot
  • git merge [branch]: merges the specified branch's history into the current one
